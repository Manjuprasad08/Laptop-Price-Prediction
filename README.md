# Laptop Price Prediction

## Project Overview
This project is about predicting the price of laptops using machine learning.
SmartTech Co. wants to know how different laptop features affect the price so they can make better business decisions.

## Problem Statement
Laptop prices depend on many factors like brand, RAM, processor, storage, and screen size.
The goal of this project is to build a machine learning model that can predict laptop prices accurately.

## Objectives
- Predict laptop prices based on specifications
- Understand which features affect laptop prices the most
- Analyze the impact of brand on laptop pricing
- Help companies position their laptops better in the market

## Dataset
The dataset contains information about laptops such as:
- Brand
- Processor
- RAM
- Storage
- Screen size
- Price

## Steps Followed
1. Data exploration and understanding
2. Data cleaning and preprocessing
3. Feature engineering
4. Model building using machine learning
5. Model evaluation and selection
6. Price prediction for new laptops

## Machine Learning Models Used
- Linear Regression
- Random Forest
- Gradient Boosting

## Key Insights
- Laptop specifications like RAM, processor, and storage strongly influence price
- Brand name also plays an important role in pricing
- The model performs well for both budget and high-end laptops

## Results
- Built a reliable model to predict laptop prices
- Identified important features affecting laptop prices
- The model can predict prices for newly released laptops

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

## Conclusion
This project helps understand laptop pricing and provides a useful prediction system.
It can be used by companies to set competitive prices and by customers to make better buying decisions.
